HTML Cheat Sheet

HTML Most Common Tags

Tag Name Description
<h1> Header 1 Use this tag to create the most important heading on your website.
<h2> Header 2 Use this tag to create the second important heading on your website. Please note that the headings are up to H6.
<ul> & <ol> unorded and numbered list Use the <ul> or <ol> tag together with the <li> to create unorded or numbered lists.
<p> Paragraph This tag defines a paragraph.
<div> Division Define an undefined division (or section) in an HTML document.

... and Their Attributes

HTML Attribute Description Example
id Identifies a HTML element in the most specific way. id="example"
class Assigns an element to one or more classes. class="cars"
style Defines the style of an HTML element and can determine the color, font, font size, etc. Please note best practice is to link a css file to style your HTML document, instead of using inline style. style="color: blue; font-size: 2em"
